## 주요 기능

### 1. 상품 관리
- 상품 등록 / 조회 / 검색
- 상품 비활성화(삭제) 및 활성화

### 2. 창고 관리
- 창고 등록 / 조회 / 검색
- 창고 비활성화(삭제) 및 활성화

### 3. 입고 관리
- 창고/상품 선택 후 입고 등록
- 입고 내역 조회

### 4. 출고 관리
- 창고/상품 선택 후 출고 등록
- 출고 내역 조회

### 5. 재고 관리
- 창고별 / 상품별 재고 현황 조회
- 창고/상품 기준 필터링
- 입출고 이력 기반 -> 자동 재고 계산

### 6. 마스터 데이터 관리
- 비활성(삭제)된 상품/창고 목록 조회
- 비활성 → 재활성화(복구) 기능
- 실제 데이터는 삭제하지 않고 소프트 삭제(Soft Delete) 방식 적용
